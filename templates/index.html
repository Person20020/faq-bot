FAQ Bot

This is the site for the FAQ Bot.

Commands

/faq

Usage: /faq [trigger]

Configuration

The responses for the bot are configured in a JSON file. Each channel that uses the bot will have its own configuration file in the config directory of this GitHub repository that has the channel ID of the channel as the filename.

Each question should have a trigger (the word that will be used in the command to get that answer), a question, and an answer.
